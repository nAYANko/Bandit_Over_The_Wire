# Level 0
To connect to bandit server through ssh,


Use the ssh command syntax

 
` ssh [user@]hostname -p [port] `
As 
` ssh bandit0@bandit.labs.overthewire.org -p 2220 `
 
Authenticate the connection by typing yes
` Type password: bandit0 `
